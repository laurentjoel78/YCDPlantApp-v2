## Design System Recommendations

### **Color Palette**

- **Primary Brand**: Green (agriculture, growth) + Blue (technology, trust)
- **Secondary**: Orange (opportunity), Red (urgency/challenges), Yellow (sunshine/hope)
- **Neutral**: Dark gray (text), Light gray (backgrounds)

### **Typography**

- **Headlines**: Bold, sans-serif (Helvetica, Arial, or similar)
- **Body**: Regular, sans-serif for clarity
- **Data Labels**: Small, sans-serif with sufficient contrast

### **Icon Style**

- **Consistency**: Use a single icon library (e.g., FontAwesome, Material Design)
- **Size**: 32-64px for dashboard elements
- **Color**: Match color palette; use iconography consistently

### **Layout Principles**

- **White Space**: Generous margins around graphics
- **Alignment**: Center-aligned for impact, left-aligned for data-heavy charts
- **Sizing**: Graphics should occupy 40-60% of section width
- **Responsive**: Ensure readability on mobile (pitch deck often viewed on tablets)

## File Delivery Specifications

### **Format**

- Preferred: SVG (scalable, editable, small file size)
- Fallback: High-res PNG (300 DPI minimum for print)
- Animated: MP4 or GIF for presentation videos

### **File Naming Convention**

```
YCD_[Section]_[GraphicType]_v[Version].svg
Example: YCD_Competitive_PositioningMatrix_v1.svg
```

### **Accessibility**

- All charts include alt-text descriptions
- Color schemes pass WCAG AA contrast requirements
- Data tables provided as backup for complex visualizations

## PDF Export Recommendations

When converting the final pitch deck to PDF with graphics:

1. **Resolution**: Set to 300 DPI for print quality
2. **Color Mode**: RGB for screen, CMYK for professional printing
3. **Fonts**: Embed all fonts to prevent substitution
4. **Images**: Compress to balance quality and file size (target: <10MB)
5. **Interactive Elements**: Convert to static images for PDF compatibility
